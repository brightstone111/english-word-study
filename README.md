# 영어 단어 학습기 (English Vocabulary Trainer)

영어 단어를 입력하면 **그림 · 영영 정의 · 발음 · 예문**을 한 화면에 보여주고,
맨 아래 **가림막**을 클릭하면 그제서야 **한글 뜻**이 공개되는 웹앱입니다.
스스로 뜻을 떠올려 본 뒤 정답을 확인하는 학습 방식에 맞춰 만들었습니다.

## 주요 기능

- 🖼️ **관련 이미지** — 단어에 맞는 사진을 자동으로 검색해 표시
- 📖 **영어 정의** — 품사 + 영영 뜻풀이
- 🔊 **발음** — 발음기호(IPA) + 스피커 버튼으로 소리 듣기 (오디오 없으면 브라우저 음성합성 대체)
- ✍️ **예문** — 단어가 쓰인 실제 문장 (단어 강조 표시)
- 👀 **한글 뜻 가림막** — 클릭해야 흐릿하게 가려진 한글 뜻이 공개
- ⭐ **즐겨찾기** — 별 버튼으로 단어 저장 (한글 뜻까지 함께 보관)
- 🕘 **최근 검색 기록** — 검색한 단어를 자동 기록, 클릭 한 번으로 재검색

즐겨찾기와 검색 기록은 브라우저 `localStorage`에 저장되어 새로고침·재실행 후에도 유지됩니다.

## 사용법

`index.html` 파일을 브라우저로 열면 됩니다. (인터넷 연결 필요)

Windows에서는 바탕화면의 `영어 단어 학습` 바로가기를 더블클릭해 실행할 수 있습니다.

## 사용한 무료 API (API 키 불필요)

| 용도 | 서비스 |
|------|--------|
| 영어 정의 · 발음 · 예문 | [dictionaryapi.dev](https://dictionaryapi.dev) |
| 예문 보충 | [Tatoeba](https://tatoeba.org) |
| 한글 번역 | [MyMemory](https://mymemory.translated.net) |
| 이미지 | [Openverse](https://openverse.org) |

- `icon.ico` — 바탕화면 바로가기 아이콘
